Privacy

SyncPad is anonymous by default, but not private by default.

  • Normal rooms do not require accounts.
  • Editable room links are anonymous room paths.
  • Anyone with the editable room path can edit that room.
  • Read-only share links hide the room name but are still secret bearer links.
  • Room text and file metadata are stored in Supabase.
  • Uploaded files are stored in Supabase Storage.
  • Browser localStorage may store drafts, preferences, theme, device name, and similar local data.
  • Encrypted room text requires the passphrase.
  • If the passphrase is forgotten, encrypted room text cannot be recovered.
  • Uploaded files are not encrypted unless file encryption is actually implemented.
  • The site owner/admin may access unencrypted rooms/files for maintenance, troubleshooting, and abuse handling.
  • Contact form submissions go through Web3Forms only if configured.
  • No analytics or ad cookies are used unless actually added to the app.

Terms

By using SyncPad, you agree to these simple terms.

  • SyncPad is a personal/portfolio project.
  • There is no guarantee, warranty, uptime promise, or SLA.
  • Do not store sensitive information, passwords, classified information, HIPAA/PII, or anything important.
  • Anyone with an editable room URL can edit that room.
  • Anyone with a read-only share URL can read that room.
  • Content may be changed or deleted by other users with access, expiration, view-once behavior, manual clear, admin cleanup, bugs, or maintenance.
  • Encrypted room text depends on remembering the passphrase.
  • Files are not encrypted unless file encryption is actually implemented.
  • The site owner/admin may remove rooms/files/reports for maintenance, security, abuse prevention, or testing.
  • The app may change or be discontinued at any time.
